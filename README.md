# trip-advisor
Travel Explorer – TripAdvisor Inspired Website
This project is a two-page responsive travel website built using HTML and CSS. It is inspired by the layout and structure of modern travel platforms. The website focuses on clean design, proper alignment, and user-friendly navigation.
🏠 Home Page
The home page includes a navigation bar, hero section, featured destinations, and a travel inspiration section. It is designed with structured layout techniques, hover effects, and responsive styling to ensure a smooth user experience across devices.
🌍 Explore Page
The explore page highlights additional destinations, travel categories, and curated experiences. It includes organized content sections and visually appealing cards to enhance readability and engagement.
The main goal of this project was to practice layout design, CSS styling, responsive design, and overall front-end development fundamentals.
